Python bindings for [QuadriFlow](https://github.com/hjwdzh/QuadriFlow) instant quad remeshing via field-guided parametrization.

## Installation

```bash
pip install pyquadriflow --find-links https://github.com/PozzettiAndrea/pyQuadriFlow/releases/latest/download/
```
